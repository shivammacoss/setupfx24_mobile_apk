# Data Safety Form Answers for Google Play Console

Use these answers when filling out the Data Safety section in Google Play Console.

## Does your app collect or share any of the required user data types?
**Yes**

## Is all of the user data collected by your app encrypted in transit?
**Yes** (SSL/TLS encryption)

## Do you provide a way for users to request that their data is deleted?
**Yes** (Users can contact support@setupfx24.com to request data deletion)

---

## Data Types Collected:

### 1. Personal Info
| Data Type | Collected | Shared | Purpose | Optional |
|-----------|-----------|--------|---------|----------|
| Name | Yes | No | Account management, KYC verification | Required |
| Email address | Yes | No | Account management, Communications | Required |
| Phone number | Yes | No | Account management | Required |

| Data Type | Collected | Shared | Purpose | Optional |

| Data Type | Collected | Shared | Purpose | Optional |

## Data Handling Practices:
- **Data is encrypted in transit:** Yes (HTTPS/SSL)
- **Data can be deleted by user request:** Yes
- **Data is processed ephemerally:** No (data is stored for account functionality)
- **Data sharing with third parties:** No user data is sold or shared with third parties for advertising
